## [564. 寻找最近的回文数](https://leetcode-cn.com/problems/find-the-closest-palindrome/)

### 说明
给定一个表示整数的字符串 n ，返回与它最近的回文整数（不包括自身）。如果不止一个，返回较小的那个。

“最近的”定义为两个整数差的绝对值最小。

提示
* 1 <= n.length <= 18
* n 只由数字组成
* n 不含前导 0
* n 代表在 [1, 10^18 - 1] 范围内的整数

### 实例
#### 1
输入: n = "123"
输出: "121"

#### 2
输入: n = "1"
输出: "0"

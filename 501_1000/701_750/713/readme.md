## [713. 乘积小于 K 的子数组](https://leetcode-cn.com/problems/subarray-product-less-than-k/)

### 说明
给你一个整数数组 nums 和一个整数 k ，请你返回子数组内所有元素的乘积严格小于 k 的连续子数组的数目。

### 提示
* 1 <= nums.length <= 3 * 10^4
* 1 <= nums[i] <= 1000
* 0 <= k <= 10^6

### 实例
#### 1
输入：nums = [10,5,2,6], k = 100
输出：8

#### 2
输入：nums = [1,2,3], k = 0
输出：0

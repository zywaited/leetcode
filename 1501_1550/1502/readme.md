## [1502. 判断能否形成等差数列](https://leetcode-cn.com/problems/can-make-arithmetic-progression-from-sequence/)

### 说明
给你一个数字数组 arr 。

如果一个数列中，任意相邻两项的差总等于同一个常数，那么这个数列就称为 等差数列 。

如果可以重新排列数组形成等差数列，请返回 true ；否则，返回 false 。

* 2 <= arr.length <= 1000
* -10^6 <= arr[i] <= 10^6

### 实例
#### 1
输入：arr = [3,5,1]
输出：true

#### 2
输入：arr = [1,2,4]
输出：false

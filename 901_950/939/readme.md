## [939. 最小面积矩形](https://leetcode-cn.com/problems/minimum-area-rectangle/)

### 说明
给定在 xy 平面上的一组点，确定由这些点组成的矩形的最小面积，其中矩形的边平行于 x 轴和 y 轴。

如果没有任何矩形，就返回 0。

* 1 <= points.length <= 500
* 0 <= points[i][0] <= 40000
* 0 <= points[i][1] <= 40000
* 所有的点都是不同的。

### 实例
#### 1
输入：[[1,1],[1,3],[3,1],[3,3],[2,2]]
输出：4

#### 2
输入：[[1,1],[1,3],[3,1],[3,3],[4,1],[4,3]]
输出：2

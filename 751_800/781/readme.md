## [781. 森林中的兔子](https://leetcode-cn.com/problems/rabbits-in-forest/)

### 说明
森林中，每个兔子都有颜色。其中一些兔子（可能是全部）告诉你还有多少其他的兔子和自己有相同的颜色。我们将这些回答放在 answers 数组里。

返回森林中兔子的最少数量。

* answers 的长度最大为1000。
* answers[i] 是在 [0, 999] 范围内的整数。

### 实例
#### 1
输入: answers = [1, 1, 2]
输出: 5

#### 2
输入: answers = [10, 10, 10]
输出: 11

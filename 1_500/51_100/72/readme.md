## [72. 编辑距离](https://leetcode-cn.com/problems/edit-distance/)

### 说明
给定两个单词 word1 和 word2，计算出将 word1 转换成 word2 所使用的最少操作数 。

你可以对一个单词进行如下三种操作：

插入一个字符
删除一个字符
替换一个字符

* 1 <= n <= 10^5
* 1 <= m <= 10^6

### 实例
#### 1
输入: word1 = "horse", word2 = "ros"
输出: 3

#### 2
输入: word1 = "intention", word2 = "execution"
输出: 5

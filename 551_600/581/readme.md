## [581. 最短无序连续子数组](https://leetcode-cn.com/problems/shortest-unsorted-continuous-subarray/)

### 说明
给你一个整数数组 nums ，你需要找出一个 连续子数组 ，如果对这个子数组进行升序排序，那么整个数组都会变为升序排序。

请你找出符合题意的 最短 子数组，并输出它的长度。

* 1 <= nums.length <= 10^4
* -10^5 <= nums[i] <= 10^5

### 实例
#### 1
输入：nums = [2,6,4,8,10,9,15]
输出：5

#### 2
输入：nums = [1,2,3,4]
输出：0

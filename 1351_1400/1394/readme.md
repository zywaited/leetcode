## [1394. 找出数组中的幸运数](https://leetcode-cn.com/problems/find-lucky-integer-in-an-array/)

### 说明
在整数数组中，如果一个整数的出现频次和它的数值大小相等，我们就称这个整数为「幸运数」。

给你一个整数数组 arr，请你从中找出并返回一个幸运数。

如果数组中存在多个幸运数，只需返回 最大 的那个。
如果数组中不含幸运数，则返回 -1 。

* 1 <= arr.length <= 500
* 1 <= arr[i] <= 500

### 实例
#### 1
输入：arr = [2,2,3,4]
输出：2

#### 2
输入：arr = [1,2,2,3,3,3]
输出：3

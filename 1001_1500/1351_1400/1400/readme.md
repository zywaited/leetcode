## [1400. 构造 K 个回文字符串](https://leetcode-cn.com/problems/construct-k-palindrome-strings/)

### 说明
给你一个字符串 s 和一个整数 k 。请你用 s 字符串中 所有字符 构造 k 个非空 回文串 。

如果你可以用 s 中所有字符构造 k 个回文字符串，那么请你返回 True ，否则返回 False 。

* 1 <= s.length <= 10^5
* s 中所有字符都是小写英文字母。
* 1 <= k <= 10^5

### 实例
#### 1
输入：s = "annabelle", k = 2
输出：true

#### 2
输入：s = "cr", k = 7
输出：false

## 331. [验证二叉树的前序序列化](https://leetcode-cn.com/problems/verify-preorder-serialization-of-a-binary-tree/)

### 说明
序列化二叉树的一种方法是使用前序遍历。当我们遇到一个非空节点时，我们可以记录下这个节点的值。
如果它是一个空节点，我们可以使用一个标记值记录，例如 #。

### 实例
#### 1
输入: "9,3,4,#,#,1,#,#,2,#,6,#,#"
输出: true

#### 2
输入: "1,#"
输出: false

#### 3
输入: "9,#,#,1"
输出: false
